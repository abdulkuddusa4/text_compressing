-To compress a file.
    1. Open up the terminal and cd to the directory where main.py lives.
    2. Then type:
            *** python main.py compress path_of_the_text_file.  ***

-To decompress a file.
    1. Open up the terminal and cd to the directory where main.py lives.
    2. Then type:
            *** python main.py decompress path_of_the_compressed_file_which_has_been_compressed_using_this_applicaton. ***

NOTE: this application only works with text files.
