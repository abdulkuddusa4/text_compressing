-To compress a file.
    1. Open up the terminal and cd to the directory where main.py lives.
    2. Then type:
            *** python main.py compress name_of_your_text_file.

-To decompress a file.
    1. Open up the terminal and cd to the directory where main.py lives.
    2. Then type:
            *** python main.py compress name_of_that_file_which_has_been_compressed_using_this_application.
